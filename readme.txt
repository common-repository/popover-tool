== Description ==

We can add popover content on selected text in wordpress block it's available on all blocks which have toolbar like paragraph block, heading block etc...

= Special Features =

* Ability to add popover on text

== Installation ==

1. Upload the entire `Popover tool` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Minimum Requirements =

* WordPress 5.0 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.
